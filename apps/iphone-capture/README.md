# iPhone Capture

The iPhone capture surface should be nearly invisible as a product.

Teacher flow:

1. Scan QR code on the Mac.
2. Take a photo or short scan.
3. Upload to the Mac.
4. Return to teaching.

Design constraints:

- No app install for MVP
- Browser-based camera capture
- Same local Wi-Fi network
- One big capture button
- Clear upload state
- No student list editing on the phone
