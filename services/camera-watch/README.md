# Camera Watch

Camera Watch is for future monitoring camera integration.

Supported source plan:

- RTSP stream
- ONVIF discovered camera
- NVR stream
- Local frame folder for testing

Operating model:

- Watch only during class windows.
- Pull frames at a controlled interval.
- Select clear face frames before recognition.
- Create observations, not final records.
- Let `AttendanceResolveSkill` decide attendance.
