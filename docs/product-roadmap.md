# Product Roadmap

## Phase 1: iPhone Assisted Attendance

Goal: prove the daily workflow with the smallest useful product.

- Mac local command center
- iPhone QR-code capture page
- Student and class database
- Manual student import
- Face enrollment photos
- Group photo recognition
- Teacher confirmation screen
- CSV/Excel export

Success metric:

- A teacher can finish attendance for a class in under 60 seconds.

## Phase 2: Fixed Camera Attendance

Goal: reduce teacher actions from photo capture to exception review.

- Mac or USB camera support
- Classroom doorway camera support
- Scheduled watch windows
- Automatic observation creation
- Late arrival detection
- Uncertain match queue

Success metric:

- The teacher reviews exceptions only.

## Phase 3: Monitoring System Integration

Goal: connect to school camera infrastructure.

- RTSP stream ingestion
- ONVIF discovery
- NVR stream configuration
- Frame selection from video streams
- Multi-camera duplicate merging
- Room-to-class schedule mapping

Success metric:

- Students walking into the room are marked present automatically during class windows.

## Phase 4: SA Skill / Robot Skill Packaging

Goal: make attendance a reusable commercial skill.

- Stable skill interfaces
- External API
- Per-school tenant settings
- Billing by active student count
- White-label install option
- Robot actor adapter

Success metric:

- The attendance node can be installed into another school's system or robot stack.
